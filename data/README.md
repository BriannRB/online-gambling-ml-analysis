# Dataset

This project uses the **Judionline Dataset** provided on Kaggle.

## Source
- Dataset: Judionline (Online Gambling Comments Dataset)
- Author: yaemico
- Link: https://www.kaggle.com/datasets/yaemico/judionline

## Description
The dataset contains Indonesian text data related to online gambling,
used for binary text classification (gambling vs non-gambling).

## Usage
Due to licensing and size considerations, the dataset is **not included**
in this repository.

To reproduce the results:
1. Download the dataset from Kaggle
2. Place the extracted files into this `data/` directory
